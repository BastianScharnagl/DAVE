# Free API Research

## Weather APIs
- **Weatherstack**: Lightweight JSON format for weather information. Part of the public-apis list.
- **weatherdbi.herokuapp.com**: Easy to implement weather API mentioned in Mixed Analytics blog.

## Cryptocurrency APIs
- **Finnhub**: Free APIs for realtime cryptocurrency data.
- **Alpha Vantage**: Offers cryptocurrency feeds along with stock market data.
- **Twelve Data**: Access to cryptocurrency market data.
- **CoinAPI.io**: Real-time & historical data across 400+ exchanges.
- **TAAPI.IO**: Technical indicators for crypto exchanges.

## Stock Market APIs
- **Finnhub**: Free realtime APIs for stock and forex.
- **Alpha Vantage**: Free stock APIs in JSON and CSV formats.
- **Twelve Data**: Stock, forex, and crypto market data APIs.
- **Alpaca**: Developer-first API for Stock, Options, Crypto Trading.
- **Finage**: Real-time and historical stock market data.

## General Resources
- **GitHub - public-apis/public-apis**: Comprehensive list of free APIs including weather, stock, and cryptocurrency.
- **Mixed Analytics Blog**: List of actually free and open public APIs with no auth needed.
